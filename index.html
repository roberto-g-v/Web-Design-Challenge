Let´s Begin:

Formula to compare = Latitude vs X

X = Temperature, Cloudiness, Wind Speed or Humidity.

Hello world, and its weather!

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, it was necessary to pull data from the OpenWeatherMap API to assemble a database on over 500 cities.

After assembling the database, from pandas we used Matplotlib to plot various aspects of the latitude vs X factors. Including temperature, cloudiness, windspeed and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.

Hypotesis to be proven:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
